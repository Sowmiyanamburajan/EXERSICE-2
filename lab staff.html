Objective of the Course

The student should be made to:

CO1: Construct Web pages using HTML/XML and style sheets (K3).
CO2: Build dynamic web pages with validation using JavaScript
objects and by applying different event handling mechanisms (K3).
CO3: Prepare dynamic web pages using server-side scripting (K3).
CO4: Make use of PHP programming to develop web applications (K3).
CO5: Develop web applications using AJAX and web services (K3).

Outcome of the Course

Upon successful completion of this course, students will:

CO1: Construct Web pages using HTML/XML and style sheets.
CO2: Build dynamic web pages with validation using JavaScript
objects and by applying different event handling mechanisms.
CO3: Develop dynamic web pages using server-side scripting.
CO4: Use PHP programming to develop web applications.
CO5: Construct web applications using AJAX and web services.
